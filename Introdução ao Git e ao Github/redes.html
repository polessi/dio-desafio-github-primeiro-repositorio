Uso o terminal de comandos para descobrir o endereço de sites famosos no Windows:

1. Aperte as teclas windows + R

2. Digite cmd e aperte ENTER

3. na tela preta que aparecerm digite ping

Será mostrado algo assim:

Programadores que trabalham com internet costumam usar banco de dados como MySql e servidores web como o Apache. A porta 3306 é usada como padrão no banco de dados MySql e a porta 80 e 443 para Apache.
